# responsive-design-with-media-queries-and-imports-on-css

Utilizando o devTools é possivel verificar que quando diminuimos o tamanho da tela ( para simular telas menores como a de um celular, tablets, entre outros dispositivos), algumas alterações são realizadas através do media query para que o usuário consiga visualizar todo o conteúdo independente do dispositivo ele esteja usando.
